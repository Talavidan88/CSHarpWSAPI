    Shenkar C# End Project - REST WS API.

    Our project was to build a a C# REST WS library. we choose to use two RESTful API's. Openweathermap, and Facebook! With our API you can easily get weather data about a location. with our Facebook API you can view your information, Post And Delete a post

    Usage And Licensing

    We believe in open source! our entire source code is free to use. If you decide to use our library instad, make sure to add Facebook and Newtonsoft json packages. our API is very easy to use.

    To Use OpenWeatherMap API

    in order to get weather data, simply create a new service IDataService weather = DataServiceFactory.GetDataService(DataServices.OPEN_WEATHER_MAP); To get data back simply use weather.GetWeatherData(new Location((int)location id, (string)location name,(string)location country,(coords)coordinates - lat and lon); you can only only insert one parameter to this function. weather.GetWeatherData(new Location(0,"RamatGan");

    To Use Facebook API

    Create a new service facebook = DataServiceFactory.GetDataService(DataServices.FACEBOOK); Authenticate facebook.Authenticate(); You can get Data by using one of the three functions: to get user details FBUserDetails facebook.GetUserDetails(); to get user likes Dictionary object - Key is page's name. facebook.GetUserLikes(); to get user events Dictionary object - Key is event's name. facebook.GetUserEvents();

    In order to post simply use: facebook.Post(string Message to post); This meathod returns the post id. to delete: facebook.Delete(string Post ID);
